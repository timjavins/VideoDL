# Triage Labels

This repo uses the default label vocabulary for the five canonical triage roles.

## Mapping

- `needs-triage` — maintainer needs to evaluate
- `needs-info` — waiting on reporter
- `ready-for-agent` — fully specified, AFK-ready
- `ready-for-human` — needs human implementation
- `wontfix` — will not be actioned

## Notes

- Use these exact strings unless the repo adopts different GitHub labels later.
- The triage workflow should not invent alternate label names.
