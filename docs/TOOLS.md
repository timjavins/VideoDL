# Tools and Dependencies

## Required runtime tools
- yt-dlp
- ffmpeg

## Backend
- Rust stable toolchain
- axum, tokio, serde, tracing

## Frontend
- Python 3.10+
- requests
- tkinter (bundled with standard Python on most platforms)

## Observability
- Backend logs via tracing
- Pollable task status endpoint
- Headless client stdout traces
